Adobe dreamweaver

o que podemos fazer com o dreamweaver

É um programa de desenvolvimento web que permite criar, editar e gerenciar sites

podemos criar pagina de comércios, aplicativos da web, consultas na web, criar páginas mestres e etc.

como trabalhamos ou fazenos

É um software de ambiente de desenvolvimento integrado (IDE), o que significa que oferece ferramentas e suportes alinhados com os objetivos dos desenvolvedores.

Explique o HTML.

html é uma linguagem de marcação de hipertexto que está presente em todos os sites na web

O que são as TAGs, descreva exemplos.

as tags são usadas para a estrutura de um site

Explique o CSS.

é um código em que você pode fazer alterações rápidas de layout, como definição de cores e fontes

Explique UTF-8 e para que usamos.

é um padrão de codificação de caracteres que representa cada caractere por 1 a 4 bytes. Usamos pra codificar caractere Unicode, representar textos em diversos idiomas, converter caracteres em uma sequencia de bits e etc.
